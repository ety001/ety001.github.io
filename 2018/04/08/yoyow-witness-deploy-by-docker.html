YOYOW 见证人教程(docker版)

发表于 2018-04-09 评论数：

前言

YOYOW 是一个基于区块链的内容激励网络，YOYOW 虽然给人第一印象是模仿 Steem，其实不然。YOYOW 在做的事情应该是一个区块链版的 UCenter。其他的事情也不多说了，不是教程范畴的，请自行去 官网 阅读 白皮书。

前置技能 和 准备工作

  • 基础的 Linux 知识
  • 基础的 Docker 知识
  • 有一台运行有 Docker 的服务器（目前 2G 内存，50G 硬盘就足够足够了）
  • 有 YOYOW 账号，且至少账号内有 11000 个币（抵押需要1w，手续费需要1k）
